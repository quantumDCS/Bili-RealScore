# Bili-RealScore
通过B站API计算BibiBibi的真实评分

# 获取剧集的mdid
1. 进入番剧播放界面
![播放页](https://github.com/quantumDCS/Bili-RealScore/blob/master/assets/%E6%92%AD%E6%94%BE%E9%A1%B5.png)
2. 点击下方番剧名称，进入详情页
3. 复制详情页网址中md后的数字
![url](https://github.com/quantumDCS/Bili-RealScore/blob/master/assets/url.png)
